# WC Design Brief — Anti-Slop Enforcement Layer

Every build starts here. No exceptions. A 90-second brief prevents a full rebuild.

---

## What AI Slop Actually Is

AI slop is not bad code. It is the absence of a point of view.

It is the statistical average of "good design" — which produces:
- Inter or Space Grotesk at 400 weight
- Purple-to-indigo hero gradient OR soft glassmorphic cards
- Centered everything, evenly spaced everything
- Three-column feature grid, rounded shadow cards
- Teal CTA button that pulses

These are technically competent and completely forgettable. The antidote is not better prompting — it is a declared, specific, defended aesthetic direction before a single line of code is written.

---

## Step 1 — Run the Brief (mandatory, takes 90 seconds)

### Q1 — What is the single job of this build?
Not "a sales app" or "a landing page." One specific job.

### Q2 — Aesthetic direction (name it specifically)
Not "modern" or "clean." A real reference with a real feel.

### Q3 — The one unforgettable element
What is the single thing someone who left the screen still remembers in 10 minutes?

### Q4 — What this explicitly is NOT
Name three specific things this build will not look like.

## Aesthetic Direction Library

### DARK MILITARY
Feel: precision instrument, war room, earned authority
- Font: Barlow Condensed 700-900 / DM Sans 400
- Color: Deep navy #0a1628, teal #58c0d0, white text, zero pastels
- WC native direction — use this as the default for all WC builds

### BRUTALIST LUXURY
Feel: expensive and confrontational simultaneously

### EDITORIAL MAGAZINE
Feel: considered, intelligent, worth reading slowly

### INDUSTRIAL DASHBOARD
Feel: functional beauty, data is the aesthetic

### RETRO FUTURISM
Feel: 1980s computer meets 2040 spaceship

### MOTION-FIRST
Feel: the interface is alive, static is not an option

### LUXURY MINIMAL
Feel: so confident it doesn't need to try

## Brief Output Format

```
DESIGN BRIEF

Job: [Single sentence]
Direction: [Name from library + 2 sentences]
Unforgettable element: [One specific thing]
NOT: [3 specific things]

Font: [Headline font] / [Body font]
Color: [Primary] / [Accent] / [Text]
Layout approach: [1 sentence]
Motion: [What moves, when, why]

Confirmed — building now.
```

## The Slop Checklist (run before delivering any build)

- [ ] Inter font anywhere? → Replace
- [ ] Purple or indigo gradient? → Remove
- [ ] Centered hero layout? → Break it
- [ ] Soft rounded glassmorphic cards on dark bg? → Remove
- [ ] Drop shadows visible on dark surface? → Remove
- [ ] More than 2 typefaces? → Consolidate
- [ ] Can name the one unforgettable element? → If no, add one
